Morse Code Converter

The Morse code, developed by Samuel Morse in the 1830s and 1840s, is a coding system that uses dots and dashes to represent characters. It was designed to enable long-distance communication using electrical signals. In the Morse code, a dot represents a short signal, while a dash represents a long signal.

Initially used in telegraph systems, the Morse code gained acceptance in the United States and later became an international means of communication. It started with encoding alphabetical characters but expanded to include numbers, punctuation marks, and other symbols.

For a long time, the Morse code played a significant role in communication. It was widely used in telegraph systems and maritime communication, and it became an important international communication tool. However, with the development of faster and more efficient communication technologies, the Morse code gradually lost its prominence.

Although the Morse code has largely been phased out in modern communication, it remains a significant milestone in the history of communication. It played a crucial role in the evolution of communication technologies and laid the foundation for wireless communication systems. Some enthusiasts, such as amateur radio hobby groups and emergency communication networks, still use Morse code.

import pygame 
import time
sentence=input("Enter the sentence what you want to convert to morse code.\n").upper()
morse_code=''
for word in sentence.split():
morse_word=' '.join(MORSE_CODE_DICT.get(char,'') for char in word)
morse_code+=morse_word+' '
print(morse_code)
def play_morse_code(morse_code):
    pygame.mixer.init()
    pygame.mixer.music.set_volume(0.2)
    def play_sound(morse):
        if morse=='.':
            pygame.mixer.music.load('dot.wav')
            time.sleep(0.1)
        elif morse=='-':   
            pygame.mixer.music.load('dash.wav')
            time.sleep(0.2)
        pygame.mixer.music.play()
        time.sleep(0.5)
    for char in morse_code:
        if char=='.' or char=='-':
              play_sound(char)

             time.sleep(0.7)
play_morse_code(morse_code)
